# 要件定義書 — 無料駐車場マップ（Free Parking Map）

**バージョン**: 1.0  
**作成日**: 2026-05-09  
**ステータス**: 確定

---

## 1. サービス概要

東京都内の駐車場付き施設を地図上に可視化し、駐車場の料金条件・施設カテゴリでフィルタリングできる家族向けお出かけ支援Webアプリ。

### ターゲットユーザー
- 子ども連れの家族（主にスマホ利用）
- 「今日どこ行こう？駐車場無料で遊べる場所は？」を探している人

### カバー地域
| フェーズ | 対象地域 |
|---------|---------|
| MVP | 東京都 |
| Phase 2 | 千葉県・埼玉県・神奈川県・茨城県 |

## 2. 機能要件

### 2-1. 地図表示（MVP）
- 東京都内の対象施設をピンで地図上に表示
- ピンタップ/クリックで施設の簡易情報ポップアップ表示
  - 施設名・カテゴリ・駐車場区分・住所・公式サイトリンク
- 地図ライブラリ：**Leaflet + OpenStreetMap**（完全無料）

### 2-2. フィルタリング（MVP）
- **駐車場区分フィルタ**（複数選択可）
  - A〜E の各区分をON/OFFで絞り込み
- **施設カテゴリフィルタ**（複数選択可）
  - 各施設カテゴリをON/OFFで絞り込み
- フィルタ結果はリアルタイムで地図・リストに反映

### 2-3. 一覧リスト表示（MVP）
- 地図と連動したサイドパネル/下部ドロワーで施設リスト表示
- 施設カード：名称・カテゴリアイコン・駐車場区分バッジ・簡易情報

### 2-4. 施設詳細ページ（MVP）
- 各施設の詳細情報ページ（URL共有可能）
  - 施設名・住所・営業時間・電話番号
  - 駐車場詳細（台数・無料時間・条件・有料時の料金）
  - 公式サイトリンク・Google Mapsリンク
  - データ取得元・最終更新日

### 2-5. キーワード検索（MVP）
- 施設名での部分一致検索

### 2-6. 現在地周辺表示（MVP）
- ブラウザの位置情報APIで現在地取得・地図を現在地に移動

## 3. 駐車場区分

| 区分 | 名称 | 定義 |
|-----|------|------|
| **A** | 完全無料 | 施設利用条件なし・完全無料 |
| **B** | 利用で無料 | 施設利用（入場・購入・利用）で完全無料 |
| **C** | 条件付き無料 | 施設利用で一定時間まで無料、以降有料 |
| **D** | 低額有料 | 有料・1時間200円以下 |
| **E** | 安価有料 | 有料・1時間201〜500円 |

## 5. データモデル

### Facility（施設）

| フィールド | 型 | 説明 |
|-----------|---|------|
| `_id` | string | Convex自動採番 |
| `name` | string | 施設名 |
| `category` | CategoryId | 施設カテゴリ |
| `address` | string | 住所（フル） |
| `prefecture` | string | 都道府県 |
| `city` | string | 市区町村 |
| `lat` | number | 緯度 |
| `lng` | number | 経度 |
| `website` | string? | 公式サイトURL |
| `phone` | string? | 電話番号 |
| `hours` | string? | 営業時間（テキスト） |
| `parkingCategory` | "A"\|"B"\|"C"\|"D"\|"E" | 駐車場区分 |
| `parkingDetails` | object | 駐車場詳細（下記） |
| `source` | string | データ取得元URL |
| `scrapedAt` | timestamp | スクレイピング日時 |
| `updatedAt` | timestamp | 最終更新日時 |
| `isVerified` | boolean | 手動確認済みフラグ |
| `isActive` | boolean | 表示フラグ |

### parkingDetails（駐車場詳細）

```typescript
{
  totalSpaces: number | null,       // 総台数
  freeMinutes: number | null,       // 無料時間（分）
  freeCondition: string | null,     // 無料条件（例: "2,000円以上購入"）
  paidRate: string | null,          // 有料時の料金テキスト
  notes: string | null,             // 備考
}
```

## 6. 非機能要件

| 項目 | 要件 |
|-----|-----|
| デバイス対応 | レスポンシブ（スマホ・タブレット・PC） |
| パフォーマンス | 地図初期表示 3秒以内（LTE環境） |
| ブラウザ対応 | Chrome/Safari/Edge 最新版 |
| 言語 | 日本語のみ（MVP） |
| アクセシビリティ | 基本的なHTMLセマンティクス・alt属性 |
| ホスティング | Vercel（無料枠） |
| 費用 | MVP段階はすべて無料枠で運用 |
